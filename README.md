# Instrucoes de uso:

1. Clonar;
2. Editar o que deseja;

# Para comitar:

1. Faca as edicoes que deseja;
2. Digite no terminal os comandos: `

```
git add .
git commit -m "Nome para seu commit aqui"
git push
```

3. Suas mudancas já devem estar disponíveis no repo!
